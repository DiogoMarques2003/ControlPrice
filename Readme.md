### Requirement
* Nodejs (v16.x) [download](https://nodejs.org/en/download/)

Cloning this repository
```
$ git clone https://github.com/DiogoMarques2003/ControlPrice.git
```

Instaling dependencies

> using **npm**
```
$ npm i
```

> using **yarn**
```
$ yarn 
```
